Der Sicherheitsberater

Erkennen Sie auf einen Blick, ob eine Webseite sicher ist. Ein grünes Häkchen zeigt, dass keine Gefahr droht. Wenn Sie versuchen, eine schädliche Webseite zu besuchen, erhalten Sie hingegen eine Warnung.

Der Spurenblocker

Schützen Sie Ihre Online-Privatsphäre. Avira Browserschutz blockiert Tracker, die versuchen, Sie auszuspähen.
Ein blaues Symbol zeigt an, dass Tracker nicht blockiert wurden. Dies kann vorkommen, wenn eine Webseite diese Tracker braucht, um richtig zu funktionieren.

Einzelne Tracker-Einstellungen

Sie können jeden Tracker einzeln steuern. Ist der Regler neben einem Tracker grau, ist der Tracker aktiv und späht Sie aus. Um ihn zu blockieren, schieben Sie den Regler nach rechts. Der Regler wird grün und der Tracker wird damit blockiert.
